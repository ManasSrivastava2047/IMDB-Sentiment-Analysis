# IMDb Sentiment Analysis using RNN and LSTM

This project performs sentiment analysis on IMDb movie reviews using Deep Learning models such as RNN and LSTM.

## Features
- NLP preprocessing
- TF-IDF vectorization
- RNN implementation
- LSTM implementation
- Accuracy and loss comparison
- Sentiment prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras
- NLTK

## Dataset
IMDb Movie Reviews Dataset

## Preprocessing Steps
- Lowercasing
- HTML tag removal
- Special character cleaning
- Tokenization

## Models Used
- Simple RNN
- LSTM

## Results
LSTM achieved better loss optimization while RNN showed competitive accuracy.
